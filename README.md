# Contador de Cliques

Este é um projeto simples em React que implementa um contador de cliques.

## Funcionalidade

- Um botão que incrementa o contador
- Exibição do número de cliques em tempo real

## Tecnologias

- React
- JavaScript

## Como executar

1. Clone o repositório:
git clone https://github.com/seu-usuario/contador-de-cliques

2. Acesse a pasta:
cd contador-de-cliques

3. Instale as dependências:
npm install

4. Rode o projeto:
npm start

## Objetivo

Praticar conceitos básicos de React como:
- Componentes
- useState
- Eventos (onClick)
